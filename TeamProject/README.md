# 📱 GadgetHub - Web Catalog Project

## 📌 Опис
GadgetHub — односторінковий веб-застосунок (SPA) для перегляду каталогу гаджетів.  
Проєкт створено з використанням HTML, CSS та JavaScript без використання front-end фреймворків.

## 🚀 Функціонал

- 📂 Каталог товарів з JSON
- 🔎 Пошук по товарах
- 🔃 Сортування (ціна ↑/↓, обрані товари)
- ❤️ Додавання в обране (localStorage)
- 🌙 Темна тема
- 📱 Адаптивний дизайн (Bootstrap)
- 🎮 Міні-гра з таймером і рекордами
- 📄 Розгортання опису товарів
- ⚡ SPA навігація без перезавантаження сторінки

## 🧩 Технології

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- LocalStorage
- JSON (як “бекенд”)

## 📁 Структура проєкту
- /data - JSON дані
- /img - зображення товарів
- /css - стилі
- /js - логіка застосунку
- index.html - головна сторінка

## 🎮 Гра
У проєкті реалізована аркадна гра:
- таймер 60 секунд
- система очок
- рівні складності
- рекорд (localStorage)
